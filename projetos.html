Sorteio Só Vitória

Aplicação web desenvolvida para realizar sorteios de forma simples e intuitiva. O projeto foi criado com foco no aprendizado de desenvolvimento Front-end, organização de componentes e experiência do usuário.

Tecnologias utilizadas: HTML, CSS, JavaScript, React, Git e GitHub.

Trisol

Projeto desenvolvido a partir de uma necessidade real apresentada por uma empresa parceira. Durante um período de colaboração, participei da elaboração e organização de informações utilizadas em um relatório técnico, aplicando conceitos de pesquisa, estruturação de conteúdo e apresentação de dados.

Além da experiência prática, o projeto representou uma oportunidade de transformar conhecimento em uma solução útil para atender uma demanda concreta.

Este projeto também marcou uma importante experiência prática, permitindo aplicar conhecimentos adquiridos em estudos e transformá-los em uma entrega real.
